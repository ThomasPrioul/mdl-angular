UI TimeWindow

Composants de sélection de plages temporelles avec calendrier intégré.

Plage de recherche

Plage sélectionnée :

{{
          timewindow() | json
        }}

Exemple de base

Sélecteur de plage temporelle avec support des périodes relatives et absolues.

Valeur sélectionnée :

{{
            selectedTimeWindow() | json
          }}

Avec période pré-définie

Exemple avec une période relative pré-sélectionnée (dernières 24 heures).

Valeur sélectionnée :

{{
            predefinedTimeWindow() | json
          }}

Utilisation

Import

import {{ '{' }} TimewindowComponent, TIMEWINDOW_STRINGS, FrenchTimewindowStrings {{ '}' }} from '@mdl-angular/ui-timewindow';

Template

<button mdl-timewindow [(ngModel)]="timeWindow" class="btn">
  Sélectionner une période
</button>

Types supportés

  • Relatif : Périodes comme "dernières 24h", "derniers 7 jours"
  • Absolu : Plages de dates fixes avec heures précises
  • Mode live : Périodes qui se mettent à jour automatiquement
